A camera-only way to help with swiping up and down for reading. 

Works on right hand only by convention so far, fingers must be tilted up. 

BONUS: Configure the apple automator in order to make a custom key binding allowing you to start and stop the program
- mine is COMMAND + CTRL + L

Tools used: Python, Hammerspoon, uv, MediaPipe, OpenCV, GitHub for VC, Apple Automator
